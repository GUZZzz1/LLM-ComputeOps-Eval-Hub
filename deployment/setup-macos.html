macOS 依赖与准备

完成这些步骤后回到部署文档启动项目。

部署文档 Web UI

1. 安装 Homebrew

如果本机还没有 Homebrew，先按官网说明安装：https://brew.sh/

brew --version

2. 安装 Python 3.11+

brew install python
python3 --version

确认版本为 3.11 或更高。

3. 安装 uv

brew install uv
uv --version

本项目推荐用 uv run ... 执行 Python 命令，避免依赖全局 Python 环境。

4. 安装 Ollama

brew install ollama
ollama --version

也可以从 Ollama 官网 下载 macOS 安装包。

5. 启动 Ollama 并拉取模型

ollama serve
ollama pull qwen2.5:1.5b
ollama list

ollama serve 需要保持运行；如果 Ollama 应用已经自动启动服务，可以直接拉取模型。
