部署文档

Day 1 本地运行准备与启动命令。

Web UI Swagger API

1. 依赖与准备

macOS

  • 安装 Homebrew。
  • 安装 Python 3.11+。
  • 安装 uv。
  • 安装并启动 Ollama。
  • 拉取默认模型 qwen2.5:1.5b。

查看 macOS 准备文档

Windows

  • 安装 Python 3.11+，并加入 PATH。
  • 安装 uv。
  • 安装并启动 Ollama。
  • 拉取默认模型 qwen2.5:1.5b。
  • 必要时调整 PowerShell 执行策略。

查看 Windows 准备文档

2. 项目启动命令

macOS / Linux

cd /Users/gsh/dev_project/LLM-ComputeOps-Eval-Hub
uv sync
uv run python scripts/init_db.py
ollama pull qwen2.5:1.5b
uv run uvicorn backend.app.main:app --reload

Windows

cd <你的项目路径>\LLM-ComputeOps-Eval-Hub
uv sync
uv run python scripts/init_db.py
ollama pull qwen2.5:1.5b
uv run uvicorn backend.app.main:app --reload

3. 启动后访问

  • Web UI: http://127.0.0.1:8000/
  • 项目文档: http://127.0.0.1:8000/docs
  • 部署文档: http://127.0.0.1:8000/deployment
  • Swagger API: http://127.0.0.1:8000/api/docs

4. 验收命令

初始化与测试

python3 scripts/init_db.py
uv run pytest -q

启动服务

uv run uvicorn backend.app.main:app --reload
