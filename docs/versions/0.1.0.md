# Version 0.1.0

`0.1.0` 是 LLM ComputeOps & Eval Hub 的 Day 1 初版，实现本地/私有 LLM 推理服务网关的最小闭环。

## 版本定位

该版本用于验证从账号、API Key 到 Ollama 推理转发和请求日志记录的基本链路。它不是完整评测平台，也不是生产级模型服务平台。

## 主要功能

- 用户注册和登录。
- 创建 API Key，完整 key 只在创建时返回。
- API Key hash 入库，仅保存 hash 和 prefix。
- 从 `models` 表读取可用模型。
- `/v1/chat/completions` 转发到本地 Ollama。
- 返回 OpenAI-compatible 风格字段，包括 `choices`、`usage`、`metrics`、`request_id` 和 `error`。
- 成功和失败请求都写入 `request_logs`。
- 用户只能查看自己的请求日志和请求详情。
- 提供 Day 1 Web UI 用于本地验证。
- 项目文档目录位于 `/docs`，Swagger 位于 `/api/docs`。

## 已验证边界

- 无 Authorization 返回 401。
- 错误 access token 返回 401。
- 错误 API Key 返回 401。
- access token 不能直接调用 `/v1/chat/completions`。
- API Key 不能调用 access token 管理接口。
- 缺少必要请求字段时返回 FastAPI validation error。
